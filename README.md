# Database Backup Tool

Automated database backup tool with scheduling and compression.

## Features

- Multiple database support (MySQL, PostgreSQL, MongoDB)
- Automated scheduling
- Compression support
- Cloud storage integration ready

## Installation

\`\`\`bash
npm install
\`\`\`

## Usage

\`\`\`bash
npm start
